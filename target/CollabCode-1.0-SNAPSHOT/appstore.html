Overview:

Have you ever wanted help with coding but aren't able to meet with someone who can help you in person? Have you ever wanted to teach someone how to code remotely? Have you ever had problems with communicating with other developers because they can't see or edit the code you are working on? Collab Code solves all of these problems and more by providing an on-line collaborative code editing tool that allows multiple people to view and edit the same document. Collab Code also provides a state of the art Chat box for users to be able to communicate with each other. In short, Collab Code will solve all of your on-line Code Collaboration needs.

Features:

  • Users login in on the log-in page with any user-name of their choice.
  • Users can then chat with each other in a state of the art chat box.
  • Users can edit the same document by using JavaScript syntax styling.
  • The Document saves any changes made to it.
  • Users need to provide their own collaborators.
